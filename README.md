* 구현 준비중

* 2024.03.09 회의내용 정리

  - 개발 주제 : 반찬가게 쇼핑몰 (**웹개발)
  
  - 개발 기간 : 반년정도 (너프하게 개발 진행할 예정)
  
  - 기능 정의

   2. 장바구니 기능 
   3. 회원가입 / 로그인 / 로그아웃
   4. 오픈api 연동 희망 (정확히 어떤 기능에 쓸지는 고민)
    - 주문배송조회 쪽을 활용할 수 있을지도....
   5. 마이페이지 (찜목록 / 주문배송조회)
